# Lesson Two
Learn how [Figwheel](https://github.com/bhauman/lein-figwheel) build ClojureScript code and how REPL
works.


## Wild

* Start REPL in Terminal
```sh
$ cd lesson_two
$ lein figwheel dev
```
* Open browser and go to ```file:///<lesson_two-dir>/resources/public/index.html```
* Now, you can REPL
