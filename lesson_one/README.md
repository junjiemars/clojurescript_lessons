## Install ClojureScript
You can install ClojureScript from Bash, use Git-Bash if you on Windows.
```sh
# install
$ HAS_CLOJURESCRIPT=1 bash <(curl https://raw.githubusercontent/junjiemars/kit/master/ul/install-java-kits.sh)

# run
$ clojurescript
```
